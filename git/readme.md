- 为什么先add 再commit
    为什么分几步
    带业务场景
    请写一个vue+langchain 聊天功能
    1.py 2.vue界面
    2个文件 add放入暂存区
    git add 1.py
    git add 2.vue
    git commit -m '' 老板要看的
